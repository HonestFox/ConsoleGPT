# ConsoleGPT
ConsoleGPT —— ChatGPT openapi based command line  interactive program
ConsoleGPT —— 基于 ChatGPT openapi 的命令行交互程序

## Usage
### 1、编译
Win:
```
make win
```

Mac:
```
make mac
```

Linux:
```
make linux
```

### 2、启动
mac & linux:
```
./bin/ConsoleGpt_linux
./bin/ConsoleGpt_darwin
```

win:
直接执行 bin\ConsoleGpt.exe
